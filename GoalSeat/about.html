About GoalSeat

GoalSeat is a student-built football ticket booking website inspired by modern ticket marketplace layouts. It focuses on presenting football matches, ticket categories, and a booking request form in a clear and structured way.

Our Goals

  • Create a clean football-themed interface
  • Show fixtures and ticket details clearly
  • Provide simple login and signup pages
  • Use HTML, CSS, and basic JavaScript effectively

What the Website Includes

The website includes a homepage, login page, signup page, and about page. It also includes navigation links, a styled table, forms, internal links, and an image map to satisfy the course requirements.
